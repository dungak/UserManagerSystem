After cloning open solution
right click on solution UserManager.API then set as startup project
Go to tools then NuGetPackageManager > Package Manager Console 
On the console ensure Default project: UserManager.API then run Update-Database
Once done Right click on UnserManagerSystem Solution then Properties.
On the opened window select multiple startup projects then select start for all projects and click apply
On the task bar click Start, it should run the API and Solution.
